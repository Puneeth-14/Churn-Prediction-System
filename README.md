# 📊 Customer Churn Prediction System

An end-to-end **Machine Learning project** that predicts whether a customer is likely to churn, along with actionable business insights.
Built with **Python, Scikit-learn, and Streamlit**.

## 🧠 Project Overview

Customer churn is a major problem for businesses.
This project uses **Machine Learning** to:

* Predict whether a customer will churn
* Provide churn probability
* Categorize customers into risk levels
* Enable bulk predictions using CSV files
* Deliver actionable insights for retention strategies

---

## ✨ Features

* 🔮 **Single Customer Prediction**
* 📂 **Bulk Prediction (CSV Upload)**
* 📊 **Churn Probability Visualization**
* 🎯 **Risk Segmentation (Low / Medium / High)**
* 💡 **Business Recommendations**
* 📥 **Download Predictions as CSV**
* 📈 **Insights Dashboard**

## 📈 Key Insights

* Customers with **low tenure** are more likely to churn
* Higher **monthly charges** increase churn probability
* Early engagement and pricing strategies can reduce churn
